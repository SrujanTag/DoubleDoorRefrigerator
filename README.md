# CompareX
Project Idea Submission: CompareX
Topic: Review & Comparison Site Niche Focus: Open-Source Developer Tools and PC
Hardware

# What Idea We Chose
We chose to build a modern review and comparison website. Our site will focus on
open-source developer tools and PC hardware. Regular review sites just show long text and
static tables. Our site will be different. It will let users interact with the data to find the exact
software or laptop parts they need.

# What We Are Going To Do
We will build advanced features to make our site the best tool for buyers.

-Custom Comparison Tables: Users can pick up to three items. The site will
instantly build a table. Users can hide prices or choose to only see the differences.
-Smart Matchmaker Quiz: Users will answer a few fast questions about their budget
and needs. The site will then give them a top 3 list of the best tools.
-Clear Trust Scores: We will be honest about how we rate things. We will show real
photos of the software or hardware in use, not just fake press photos.
-Saved Workspaces: Users can make an account. They can save their favorite tools
to a list and share that list with their team.
-Fast Summary Cards: We will put a short box at the top of every review. It will have
a two-sentence summary, the best three features, and the worst three features.

# How We Are Going To Implement It
We will build this project in clear steps to make sure everything works perfectly.

-Database Setup: We will create a database to hold all the details, prices, and
features for the tech products.
-Web Interface (Frontend): We will build clean, fast web pages. We will design
custom buttons and sliders for the interactive comparison tables and the matchmaker
quiz.
-Logic and Search (Backend): We will write code to sort the database fast. This
code will power the "Alternative To" search bar and the matchmaker quiz results.
-User Accounts: We will add a secure login system. This will let users save their
favorite product lists and get price drop alerts safely.
-Testing: We will test the site to make sure the tables load fast and the quiz gives the
right answers before we launch.
