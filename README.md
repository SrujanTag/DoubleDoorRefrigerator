# SpendWise 
Smart Expense Tracking & Group Splitting App

SpendWise is a comprehensive web application designed to help individuals and groups manage their finances effectively. It combines the simplicity of a personal budget tracker with the collaborative power of a bill-splitting tool.

# Features
Personal Dashboard: Track daily spends and categorize income/expenses.

Group Split: Create groups, add friends, and settle debts easily (similar to Splitwise).

Smart Tips: AI-driven or rule-based suggestions to help users save money.

Data Visualization: Interactive charts to see where your money goes.
