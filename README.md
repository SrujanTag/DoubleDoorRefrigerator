# 🚀 CompareX  
**Interactive Review & Comparison Platform for Developer Tools & PC Hardware**

---

## 📌 Overview  
CompareX is a modern web-based platform designed to help users **compare open-source developer tools and PC hardware efficiently**.  

Unlike traditional review websites with static content, CompareX provides **interactive and dynamic comparisons**, allowing users to quickly find the best tools or hardware based on their needs.

---

## ✨ Features  

### 🔍 Custom Comparison Tables  
- Compare up to **3 products/tools at once**  
- Options to:
  - Hide pricing  
  - Show only differences  

### 🧠 Smart Matchmaker Quiz  
- Answer a few quick questions based on:
  - Budget  
  - Use-case  
- Get **top 3 personalized recommendations**  

### ⭐ Transparent Trust Scores  
- Clear and honest rating system  
- Real usage visuals instead of promotional images  

### 💾 Saved Workspaces  
- User accounts to:
  - Save favorite tools  
  - Create comparison lists  
  - Share with team members  

### ⚡ Fast Summary Cards  
Each review includes:  
- Short 2-line summary  
- ✅ Top 3 advantages  
- ❌ Top 3 drawbacks
